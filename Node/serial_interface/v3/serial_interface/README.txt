---------------------------------------------------------------------ARDUINO--------------------------------------------------------

I: Go to 'addiprograms/c++/', select a project you want;

II: open the '.ino' file, and upload the sketch to your arduino;

---------------------------------------------------------------------NODE-----------------------------------------------------------

I: Open cmd and navigate to the 'User/your usenname' folder;

II: install the required packages:

	1. npm i request
	2. npm i --save-dev electron-rebuild
	3. npm i serial
	4. npm i -g json-server

III: go to the 'server' folder and start 'server.bat';

IV: go back to the 'serial_interface' folder, and start 'communicate.js';

---------------------------------------------------------------------WEB/CLIENT------------------------------------------------------


Go to 'addiprograms/web/', select a project you want or use the library to create your own.
